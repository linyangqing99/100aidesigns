# 100 AI Designs

100 runnable product-design studies made with AI, each documented as a reusable design language.

The project is inspired by the discovery and distribution model of [21st.dev](https://21st.dev), but focuses on complete small products rather than isolated UI components. All product concepts, implementations, prompts, and analysis in this repository are independently produced unless a design manifest says otherwise.

## Current status

- 001 Lumen Micro Journey — live
- 002–100 — planned through a 10-family research matrix

See `docs/UI-SYSTEM.md` for the enforced site-wide interface contract, plus `docs/ARCHITECTURE.md` and `docs/ROADMAP-100.md` for the product structure and release plan.
